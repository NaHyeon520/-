# Electric Piano Project

<ul>
  <il>Number buttons: Music scale</il>
  <il>DIP switch: Play recorded music. DIP1 -> original speed, DIP2 -> *2, DIP3 -> *4, DIP8 -> starts recording when it's on</il>
</ul>
